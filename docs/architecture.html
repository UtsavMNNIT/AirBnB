Storage & search

PostgreSQL

Listings, bookings, users. Primary + read replicas, partition/shard by region.

Redis

Session, hot listing cache, availability locks, rate-limit counters.

Elasticsearch / OpenSearch

Full-text + geo search, faceted filters; fed from Kafka.

Object storage (S3)

Original photos; origin for the image CDN.

Warehouse / Lake

Snowflake/BigQuery for analytics + ML training.

Tracing, metrics, logs, RUM/Core Web Vitals, error tracking, SLOs.

Security & secrets

WAF, secrets manager, PCI scope isolation for payments.
